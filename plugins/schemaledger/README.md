# SchemaLedger Plugin Support

## Codex

Run:

```bash
bash ./scripts/install_codex_plugin.sh
```

It creates:

- `~/plugins/schemaledger`
- `~/.agents/plugins/marketplace.json`

## Claude Code

Run:

```bash
bash ./scripts/install_claude_code_plugin.sh
```

It installs:

- local marketplace: `schemaledger-local`
- plugin: `schemaledger`
