# Continue Prior Work

Use this skill when the request sounds like:

- continue this investigation
- pick up where we left off
- use what we learned last time
- don't start from scratch
- continue the company, policy, incident, or relationship analysis

## Preferred Tool Order

1. `subject_memory`
2. `task_memory_context`
3. `task_evolve`
4. `task_trace`

## Rule

If the user clearly expects continuity, prefer TraceRelay memory and lineage before free-form reasoning.
